Agave Docker Container Runner Demo

This is a simple front-end webapp demostrating Agave's support for running Docker compute containers.

This demo is meant for evaluation purposes, kicking the tires, taking it for a spin around the block, trying it on for size.

See how I didn't use the words production, real work, mission critical, etc. That's because this demo is backed by a enough compute power to get you around the block one or two times. Trying to do much more than that will cause the server to get very sad at you and, more than likely, pack up its containers and go home.

Not to worry, though. Stick to the happy path of this demo and you'll get a good idea of how easy it can be to run Docker containers with Agave. Once you're ready to take on something a bit meatier, you can point Agave at your own Docker host(s) and cut loose.

Getting started

To use this webapp:

  1. Login so you can access your apps and data.
  2. Select the compute container you want to run
  3. Fill in the job request form
  4. Specify your data.
  5. Press sumbit to run your job
  6. Monitor the progress in the right pane

Further reading

For more information about accessing your own systems with Agave, please visit the App Management Tutorial on the Agave Developer's site.

For more information about Reproducible Science with Agave and Docker, please visit the Agave blog

You can download the full source code for this example app and client application from the Agave Docker Runner Webapp Demo Github repository. You can also view the live app on the github pages site.
